# Шпаргалка по Git

### Просмотр истории

Показать историю коммитов:

```bash
git log
```

### Работа с файлами и ветками

Скопировать файл из другой ветки в текущую:

```bash
git checkout <branch> -- <file>
```

### Работа с тегами

Показать список всех тегов в репозитории:

```bash
git tag -l
```

### Работа с ветками

Удалить локальную ветку:

```bash
git branch -d <name>
```

Показать список всех веток (локальных и удалённых):

```bash
git branch -a
```
